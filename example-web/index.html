How Fuzzy Matching Works
Pass 1 — Wildcard regex match:
Pattern add * to todo becomes regex /^add (.+) to todo$/. If input matches exactly, confidence = 1.0 and * is the extracted text.

Pass 2 — Dice similarity (when threshold > 0):
Dice = (2 × |intersection|) / (|A| + |B|)

Example: Input = "please add milk to todo list" (6 words)
Pattern = "add * to todo" → literal words: {add, to, todo} (3 words)
Intersection = {add, to, todo} = 3 matches
Dice = (2 × 3) / (6 + 3) = 0.67

At threshold 0.5 → match. At 0.8 → no match.
The formula penalizes filler words, so "add milk to todo" scores higher (0.86) than "please could you add milk to my todo list" (0.50).
